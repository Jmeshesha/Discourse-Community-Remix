# Discourse Communities in Game Development

The concept of a discourse community is incredibly useful topic that can be used to develop features games that more align with what a player would want. In this article, I hope to explain what a discourse community is, why the gaming community is a discourse community, and how it could be used in game development.

## What is a Discourse Community

There are [a lot of defintions](Borg.pdf) of a discourse community. I define it as any group of people who communicate using words specific to the group in order to achieve some shared goals.
