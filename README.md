# Discourse Communities in Game Development

Discourse communities are an incredibly useful topic that can be used to develop features in games that more align with what players want. In this article, I hope to explain what a discourse community is, why the gaming community is a discourse community, and how it could be useful when you develop games.

## What is a Discourse Community

The topic of discourse communities originated in academic studying of writing. The exact definition of a discourse community is [not agreed upon](Borg.pdf) and I think its definition depends on the context. For developing software in general, I think a useful definition of a discourse community is any group of people who communicate using words specific to the group in order to achieve some shared goals.
